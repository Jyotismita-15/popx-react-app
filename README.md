A React.js implementation of the PopX mobile application UI based on the provided Adobe XD design.

## Tech Stack

* React 18
* React Router DOM v6
* CSS3
* Functional Components & Hooks

## Features

* Pixel-perfect UI implementation based on the Adobe XD design
* Mobile application interface centered on the webpage
* Seamless navigation using React Router
* User registration and login flow
* Local Storage based user data persistence
* Account Settings page
* Sign Out functionality
* Responsive and clean user interface

## Getting Started

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

Open:

```text
http://localhost:3000
```

in your browser.

## Build for Production

```bash
npm run build
```

## Deployment

The project can be deployed using:

* Vercel
* Netlify
* Any static hosting platform supporting React applications

## Assignment Objective

Build a React.js application matching the provided Adobe XD design while ensuring clean code structure, proper navigation, and a mobile-centered user experience.
